# Anttuii

A modern, mouse-friendly terminal app for macOS.

## Features

- **Tabbed interface** - Browser-style tabs with + button
- **Smart completions** - Command suggestions with shell history integration
- **File browser sidebar** - Navigate files with git status indicators
- **File preview** - Quick preview with git diff comparison
- **Image preview** - View images directly, compare git-modified versions

## Requirements

- macOS 14+
- Xcode 15+

## Build & Run

```bash
swift build
swift run AnttuiiClient
```

## Tech Stack

- SwiftUI + AppKit
- SwiftTerm for terminal emulation
